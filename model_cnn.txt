Model: AnchorFreeCNNStudent (CNN + FPN + FCOS Head)

Input:
- Image tensor: 3 x 512 x 512

Backbone: SimpleCNNBackbone
- Preprocessing Stages:
  - S1: ConvBNReLU(3 -> s1, k=3, s=2)
  - S2: ConvBNReLU(s1 -> s2, k=3, s=2)
- C2 (stride 4): output of S2
- C3 (stride 8):
  - ConvBNReLU(s2 -> c1, k=3, s=2)
  - ResidualBlock(c1)
- C4 (stride 16):
  - ConvBNReLU(c1 -> c2, k=3, s=2)
  - ResidualBlock(c2)
- C5 (stride 32):
  - ConvBNReLU(c2 -> c3, k=3, s=2)
  - ResidualBlock(c3)
  - SpatialPyramidPooling(c3) [maxpool k=3/5/7, stride=1], concat + 1x1 conv + BN + ReLU

FPN: SimpleFPN
- Lateral 1x1 convs on C2/C3/C4/C5 -> P2/P3/P4/P5
- Top-down: P5 -> P4 -> P3 -> P2 (nearest upsample + add)
- 3x3 conv on P2/P3/P4/P5
- Extra levels:
  - P6 = Conv3x3(stride=2) on P5
  - P7 = Conv3x3(stride=2) on ReLU(P6)

Context Gate (GlobalContextGate):
- GAP on C5 -> FC(ReLU) -> FC -> sigmoid gate (size = FPN channels)
- Gate applied to each P level
- BatchNorm after gate per P-level

Head: AnchorFreeHead (FCOS-style, depthwise separable)
- 3x depthwise-separable conv layers per tower (cls/reg)
- Outputs per level:
  - cls_logits: [B, H*W, num_classes(no-bg)]
  - bbox_regression: [B, H*W, 4] (l,t,r,b distances)
  - centerness: [B, H*W, 1]
- Dropout2d on FPN features before head

Loss:
- Classification: Focal Loss (sigmoid)
- Regression: GIoU Loss
- Centerness: BCEWithLogits

Post-process:
- score = sigmoid(cls) * sigmoid(centerness)
- Score thresholding
- pre_nms_topk
- Batched NMS (class-aware)
- max_detections

- student_cnn_base_channels = [24, 48, 96]
- student_cnn_fpn_channels = 64

- student_cnn_head_depthwise = true

- student_cnn_fcos_ranges = [[0, 32], [32, 64], [64, 128], [128, 256], [256, 512], [512, 1e8]]
- student_cnn_s_min = 0.03
