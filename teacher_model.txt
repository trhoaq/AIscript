Teacher Model Changes (applied from INSTRUCT.md)

1. Anchor scales
- Lowered s_min for small objects: student_cnn_s_min = 0.03 (consider 0.02 if objects are ~15–20px)
- Added aspect ratio 1.0 at all pyramid levels: [1, 2] / [1, 2, 3]

2. FPN levels
- Added P2 (stride 4) to improve small object detection
- FPN now outputs P2–P7 (6 levels)

3. Regression loss
- Replaced SmoothL1 loss with GIoU loss for bbox regression

4. Global context gate
- Gate applied after FPN
- BatchNorm after gate per pyramid level for stability

5. CNN backbone/FPN alignment (student)
- S1/S2 preprocessing (stride 2 each), C2/C3/C4/C5 with strides 4/8/16/32
- Single SPP at C5 before FPN

Notes:
- Teacher model code (SSDMobile) now uses GIoU for regression.
- Student model is anchor-free (FCOS-style) with centerness head.
